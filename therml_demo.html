What is THERML?

THERML is a library for sampling from binary probabilistic graphical models using energy-based models. The probability distribution is:

P(x) ∝ e-βE(x)

where E(x) is the energy and β is inverse temperature.

This demo shows the simplest model: an Ising model with binary spins {-1, +1}.

Key Concepts

  • Binary Spins: Each variable can be +1 (up/red) or -1 (down/blue)
  • Energy Function: E(x) = -Σ(biasi × xi) - Σ(weightij × xi × xj)
  • Gibbs Sampling: Update each spin based on its neighbors (hardware-friendly!)
  • Temperature: Higher β = more deterministic, lower β = more random
  • Graph: This uses a simple chain topology (each spin connected to neighbors)
